# Audio Assets

Stage 12 introduces a first-pass audio layer for MVP playtesting.

- `sfx/`: short synthesized WAV cues generated inside this project.
- `music/tensefutureloop.ogg`: current default BGM from OpenGameArt, selected because it is actual music-style sci-fi loop audio and Godot imports it cleanly.
- `music/dark_things_loop.mp3`: downloaded music candidate; not used by default because Godot currently marks the MP3 import as invalid in this project.
- `music/ball_logic_pulse.mp3`: rejected pulse-style track kept as a fallback/reference.
- `music/finite_core_clean_pulse_loop.wav`: project-generated clean dark pulse loop kept as a fallback/reference.
- `music/finite_core_dark_radar_loop.wav`: older project-generated noisy dark radar loop kept as a fallback/reference.
- `music/electronic_music_loop.ogg`: brighter CC0 electronic loop kept as a fallback/reference.
- `music/project_utopia.ogg`: older ambient loop kept as a fallback/reference.
