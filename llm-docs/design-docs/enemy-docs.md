设计敌人的生态系统和寻路AI是让自动化游戏充满生机与压迫感的关键。你的思考非常深入，特别是关于“如何防止敌人被打散（添油战术）”的疑问，这正是许多RTS游戏底层架构中最难解决的痛点之一。

以下是为你量身定制的敌人设计文档及底层机制解决方案。

### 一、 核心生态：刷新机制与资源消耗模型

为了让敌人的进攻具有合理的节奏，并且与玩家的“自动化生产”产生化学反应，建议采用类似《Factorio》与《戴森球计划》的“威胁值（Threat）/ 污染吸收”混合模型。

1. **敌对巢穴的资源模型**
敌方巢穴并非凭空无限刷怪，而是拥有自己的“经济系统”。巢穴会吸收玩家工厂产生的“威胁值”（例如：机器人的电磁噪音、工厂的耗电量或开采矿物时的震动）作为能量。
2. **防守与进攻的预算分配**
巢穴会利用这些吸收来的能量（预算）进行两种生产：
* **本地防守部队（被动刷新）**：巢穴会消耗极少的资源维持一定数量的防守单位在基地附近游荡。当玩家主动靠近时，它们会发起攻击。
* **突袭小队（主动进攻）**：当巢穴积累了足够的“威胁预算”后，它会“购买”不同等级的兵种，一旦小队集结完毕，就会向玩家的污染源或核心基地发起进攻。



### 二、 核心技术难题：如何防止敌人被打散？

在巨大的地图上，如果只给每一个生成的敌人下达“攻击玩家基地”的指令，由于地形阻挡和不同单位移速不同，它们最终会变成排成一字长蛇阵的“葫芦娃救爷爷”，这会让玩家的防线毫无压力。

为了保证你精心设计的敌人组合（例如“肉盾+远程+奶妈”）能够作为一个整体压境，你需要引入“集结机制”**结合**“Boids（群落/鸟群）集群算法”：

1. **集结点（Rally Point）机制**：巢穴生成兵力后，不会立刻让它们进攻，而是在巢穴外围设定一个集结点。直到该波次的敌军数量达到阈值（例如积攒了10只），才会整体切换为“进攻状态”。
2. **小队领袖与 A* 寻路**：不要为这10只怪物分别计算前往玩家基地的 A* 复杂路径。你只需要在逻辑上指定一只处于中心的怪物作为“小队长”（Leader），或者生成一个隐形的引导点，仅为这个领袖计算跨越宏观地图的路径。
3. **Boids 集群行为**：小队中的其他成员不需要进行长距离寻路，而是根据 Boids 算法跟随小队长移动。只需为它们编写三个简单的向量规则：
* **凝聚力（Cohesion）**：单位会试图向周围同伴的平均中心点移动，保证队伍不脱节。
* **对齐（Alignment）**：单位会调整自己的朝向和速度，使其与周围同伴保持一致。
* **分离（Separation）**：当单位之间靠得太近时，会产生排斥力，防止所有怪物重叠穿模。
通过这套方案，你的怪物大军就能像真正的纪律部队一样，抱团且阵型完美地推进到玩家基地。

### 三、 敌方科技树与兵种设计文档

敌人分为四个阶段，随着游戏时间推进或玩家摧毁的敌方巢穴数量增加而逐步解锁（类似进化机制）。

#### 1. 前期 (Early Game) - 清理与扩张阶段

* **核心设定**：玩家刚刚建立矿区，敌人主要表现为领地被侵犯的“野生机械兽”。
* **基础兵种：拾荒猎犬 (Scrap Hound)**
* *行为模式*：主要在巢穴周围巡逻防守，受到微量污染刺激时会组成3-5只的小队袭击玩家边缘矿机。
* *战斗能力*：移动速度极快，只造成轻微的【动能伤害】。血量极低，玩家使用基础机枪即可轻松应对。


* **掉落物**：基础齿轮、初级感应线圈（用于玩家合成基础的自动射击逻辑芯片）。

#### 2. 中期 (Mid Game) - 阵地拉锯阶段

* **核心设定**：玩家的工厂开始产生大量电磁噪音（威胁值），敌方巢穴升级并开始有组织地拔除玩家的生产线。
* **组合兵种 A：重甲犀牛 (Armored Rhino)**
* *战斗能力*：极高的【动能护甲】，免疫击退。只造成极高的【近战动能伤害】。负责在队伍最前方吸收玩家机枪塔的火力。


* **组合兵种 B：等离子飞蚊 (Plasma Mosquito)**
* *行为模式*：躲在犀牛后方输出。一旦犀牛死亡，它们会试图向后拉扯。
* *战斗能力*：发射低频段的【能量伤害】光束，专门克制玩家堆叠的重型履带底盘。


* **掉落物**：高频振荡器、能量传导液（玩家推进科技树，制造“激光发射器”与“热量控制逻辑芯片”的关键材料）。

#### 3. 后期 (Late Game) - 逻辑与战术博弈阶段

* **核心设定**：敌方开始出现具有“反制逻辑”的特种单位，逼迫玩家重新编写自己机器人的 If-Then 规则。
* **特种兵种：信号干扰塔 (Jammer Construct)**
* *行为模式*：移动极其缓慢，永远处于敌方群落的正中央，受到所有敌军的严密保护。
* *战斗能力*：本身没有伤害，但会散发巨大的 EMP 光环。任何进入光环范围的玩家机器人，其配置的第1条和第2条 If-Then 逻辑将被强制屏蔽。
* *设计意图*：由于基础寻敌逻辑被屏蔽，玩家必须专门生产一种只装配“狙击导弹”且写入 `If 发现干扰塔 -> Then 远距离锁定打击` 的特种机器人来破局。


* **掉落物**：量子处理器（用于解锁玩家机器人的高阶动作指令，如“集火”、“掩护”）。

#### 4. 大后期 (End Game) - 终极防御战

* **核心设定**：敌对的“主脑”巢穴感到致命威胁，开始倾巢而出，生产极为昂贵的终极单位。
* **首领兵种：自适应巨兽 (Adaptive Behemoth)**
* *行为模式*：主动锁定玩家的“工厂核心”直线推进，摧毁沿途的所有传送带和建筑。
* *战斗能力*：拥有【动能】与【能量】双重护盾。当玩家对其造成超过一定阈值的某种伤害时（例如一直用激光射击），巨兽会自动生成对应的“能量免疫护甲”。
* *设计意图*：主模块单一的玩家将被完全克制。玩家必须编写出精妙的协同逻辑：`If 巨兽能量护甲开启 -> Then 激光部队停止射击 & 动能炮部队开始轰炸`。


* **掉落物**：主脑核心（通关游戏或解锁无限建造沙盒模式的关键道具）。
